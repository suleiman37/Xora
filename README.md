# 🚀 Xora Landing Page

Xora is a modern landing page built with **React + Vite + Tailwind CSS**.
It focuses on clean UI, responsiveness, and high performance.

---

## 📸 Preview

---

## ✨ Features

* ⚡ Fast performance (powered by Vite)
* 🎨 Clean and modern UI design
* 📱 Fully responsive (mobile, tablet, desktop)
* 🔄 Smooth scrolling (react-scroll)
* 🧩 Reusable components
* 📁 Well-structured project

---

## 🛠️ Tech Stack

* React
* Vite
* Tailwind CSS
* JavaScript (ES6+)

---

## 📂 Project Structure

```
src/
│
├── assets/        # images, icons, data
├── components/    # reusable components
├── sections/      # page sections (Hero, Features, FAQ...)
├── App.jsx
└── main.jsx
```

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/xora.git
cd xora
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
